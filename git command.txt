git init 初始化git
git pull [--rebase] [origin] [branch] [options] = git fetch [origin]  =[branch]+ git merge FETCH_HEAD + git rebase [origin] [branch]
拉取远程仓库
git push [origin] [branch] 将本地分支推送到远程分支
git status 查看当前git状态
git add [.或/:全部，路径/：某个文件夹里的全部文件，等] 添加缓存文件
git commit -m/-a/-am message（字符串，引号包围）提交缓存文件到分
git branch [brachName] 以branchName为名称创建本地分支
git checkout [brachName] 切换当前分支
git merge [brachNames] 合并分支到当前分支
git clone [origin] 克隆默认分支
git clone [origin] -b [branch] 克隆对应分支

软撤销
git reset --soft HEAD~1
表示撤销最后一次的 commit ，1 可以换成其他更早的数字


硬撤销
本地代码会直接变更为指定的提交版本，慎用
删除工作空间改动代码，撤销 commit，撤销 git add .
注意完成这个操作后，就恢复到了上一次的commit状态。
git reset --hard HEAD~1
git reset --sort HEAD~1

git reset --hard [git索引号/版本号]
还原到某次提交，可以咋.git文件下的logs文件夹找到HEAD文件，打开后可检索每次提交的git索引号/版本号

git config --global http.sslVerify false/true 禁用/启用SSL验证
